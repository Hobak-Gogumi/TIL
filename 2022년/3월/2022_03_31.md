#### numpy
+ Numeric Python의 약자
+ np 배열 생성 - `array()` 함수 사용
```python
import numpy as np

a = [1, 2, 3, 4]
b = [4, 5, 6, 7]

numpy_a = np.array(a)
numpy_b = np.array(b)

sum = numpy_a + numpy_b
# sum = [5, 7, 9, 11]
# 리스트와의 차이점. 리스트는 합 연산을 하면 연결이 됨. numpy 배열은 연산을 한다
```
+ numpy 배열에서 사용되는 속성 - dtype
```python
import numpy as np
arr1 = np.array([[1.0, 2, 3], [4, 5, 6]], dtype=np.int32)

print(arr1.dtype) # int32
```
