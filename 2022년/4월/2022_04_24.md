# 2022.04.24(일)

#### 장고 MTV 패턴
+ Model-Template-View
+ Model(모델)
    - MVC 패턴의 `모델에 대응`
    - DB에 저장되는 데이터 의미
+ Template(템플릿)
    - MVC 패턴의 `뷰에 대응`
    - 장고에서는 뷰에서 로직을 처리한 후 html 파일을 context와 함께 렌더링 하는데, 이 때의 html 파일을 템플릿이라 칭함
+ View(뷰)
    - MVC 패턴의 `컨트롤러에 대응`
    - 요청에 따라 적절한 로직을 수행하여 결과를 템플릿으로 렌더링하며 응답
