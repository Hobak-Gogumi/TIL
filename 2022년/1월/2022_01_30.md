#### 파이썬 출력문
+ print()문은 기본적으로 줄 바꿈
+ end 옵션을 사용하면 줄 바꿈을 안 할 수 있음
```
print("줄")
print("바꿔줘") # 각기 다른 줄로 출력됨
```
```
print("줄", end="")
print("바꾸지마!") # 같은 줄로 출력됨
```
+ 다른 문자 또는 문자열을 넣으면 구분자처럼 작동한다
+ 파이썬에서는 기본적으로 끝에 \n이 들어가있고, end=""을 쓰면 \n을 지워주는 원리.

#### 파이썬의 실수형
+ 파이썬 실수형 기본은 **float**
